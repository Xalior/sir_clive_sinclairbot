# 🎮 sir_clive_sinclairbot

A Discord channel-manager bot — with a web dashboard for admin. Built for the ZX Spectrum Next community. 🇬🇧🕹️

## Tech Stack

- **Discord.js** — bot framework
- **Express** — web dashboard
- **Redis** — session & state
- **TypeScript** — because types are nice

## Setup

```bash
pnpm install
cp env.sample .env   # fill in your tokens
pnpm dev
```
