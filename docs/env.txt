BOT_TOKEN
=========
Discord Application Token

OPENAI_TOKEN
============
OpenAI compatible API Access Token

CACHE_URL
=========
REDIS URL - in the following format:
    redis://host:6379/dbid

RELAY_SIGNING_KEY
=================
HMAC signing key for the relay plugin (org.xalior.relay).
Required — bot will refuse to start without it.
Rotate by editing .env and restarting; senders must use the same value.

RELAY_CLOCK_SKEW
================
Clock-skew tolerance window for relay HMAC requests, in seconds.
Optional — defaults to 30. Requests with timestamps outside ±this
window are rejected.
